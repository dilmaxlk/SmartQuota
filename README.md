<h1>SmartQuota</h1>

<h2>OpenSource Quotation Management System. Online Quotation, Invoice, and Job Management System for any business.</h2>

<p><br />
<strong><span style="font-size:20px">Features</span></strong></p>

<ul>
	<li>Responsive user interface.</li>
	<li>Add unlimited customers.</li>
	<li>Add unlimited quotations and quotation items.</li>
	<li>Add unlimited jobs.</li>
	<li>Generate an invoice for a related quotation.</li>
	<li>Report dashboard.</li>
	<li>Add unlimited system users and control their access level.</li>
</ul>

<p>&nbsp;</p>

<p><span style="font-size:20px"><strong>Minimum System Requirements</strong></span></p>

<ul>
	<li><strong>PHP Version 7.4</strong></li>
	<li><strong>MySQL Version 5.x or higher</strong><br />
	&nbsp;</li>
</ul>
